Le tri par slection est l'un des plus simples algorithmes de tri. Il consiste :

  • parcourir les $n$ valeurs trier,
  • identifier la plus petite des valeurs,
  • dplacer cette valeur sa position dfinitive dans le tableau tri, c-a-d. en premire position,
  • recommencer pour identifier et dplacer la seconde plus petite des valeurs,
  • et ainsi de suite pour toutes les valeurs.

A l'itration $i$ ($1 \le i < n$), les $i$ plus petites valeurs sont correctement places dans un ordre tri, c-a-d. dans les $i$ premires cases du tableau. A la fin, les $n$ valeurs sont correctement places, c-a-d. de faon trie par ordre croissant.

Exemple. Le tableau t0 = [2, 4, 1, 3] est transform successivement en :

  • [1, 4, 2, 3] : 1 est la plus petite valeur de t0, elle est donc plac l'indice 0,

  • [1, 2, 3, 4] : 3 est la plus petite valeur suivante, elle est donc plac l'indice 2.

def tri2D_selection(t: list[list[IntOrStr]], n: int, p: int, num_col: int) -> list[list[IntOrStr]]:
    '''tri par selection d'un tableau 2D d'int ou de str
    selon l'ordre croissant de num_col'''
    assert n == len(t)
    for _ in range(n):
        assert p == len(t[_])
    assert 0 <= num_col < p or -p < num_col <= -1
    
    for i in range(n):
        min_indx=i
        for j in range(i+1,n) :
            if t[j][num_col] < t[min_indx][num_col] :
                min_indx=j
        for k in range(p) :
            c=t[i][k]
            t[i][k]=t[min_indx][k]
            t[min_indx][k]=c
        
    return t

Elment majoritaire : sens absolu. L'lment majoritaire d'une liste de n lments est, lorsqu'il existe, l'lment qui apparat strictement plus que n//2 fois.

def distM(i: int, j: int, cols: tuple[int],
          t: list[list[IntOrStr]], n: int, p: int) -> int:
    '''retourne la distance de Manhattan entre les vecteurs des colonnes cols 
    des lignes i et j du tableau t de taille n x p.'''
    assert n == len(t)
    for _ in range(n):
        assert p == len(t[_])
    assert 0 <= i < n or -n <= i < -1
    assert 0 <= j < n or -n <= j < -1
    distance = 0
    for k in cols:
        distance += abs(t[i][k]-t[j][k])
    return distance

def lirePoudlard(nomDuFichier: str) -> list[list[IntOrStr]]:
    '''lit nomDuFichier au format csv et retourne un tableau 2D'''
    res = []
    with open(nomDuFichier, 'r', encoding="utf-8") as donnees_csv:
        eleves = csv.reader(donnees_csv, delimiter=";")
        eleves.__next__() # saut ligne 0
        for e in eleves:
            val_e = (str(e[0]), int(e[1]), int(e[2]), int(e[3]), int(e[4]), str(e[5]))
            res.append(list(val_e))
    return res
